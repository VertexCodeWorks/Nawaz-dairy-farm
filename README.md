# Nawaz Dairy Farm (React + Tailwind)

Modern responsive website for Nawaz Dairy Farm.

## Setup

1. Open terminal in `c:\Users\ahame\WorkSpace\NawazDairyFarmReact`
2. Run `npm install`
3. Run `npm run dev`
4. Open URL shown (usually http://localhost:5173)

## Features implemented

- React.js (functional components + hooks)
- Tailwind CSS styling
- Sections: Hero, Products, About, Service Area, Contact, Footer
- WhatsApp order integration with dynamic encoded message
- Form validation and feedback
- Responsive layout for mobile/tablet/desktop
- Sticky top navigation
- Placeholder images from Unsplash
- Map integration + directed Google Maps link

## WhatsApp ordering

- Uses WhatsApp number `+91 91828 79423`
- Pre-fills order data: Name, Phone, Address, Location, product quantities.
- Opens `https://wa.me/919182879423?text=<encoded_message>`
